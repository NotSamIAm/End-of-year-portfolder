HARDWARE OF PCs.

Mother Board(MOBO): This piece of hardware is the source of all computing power in the pc. this is a base board, which all other hardware is part of or connected to.

Central Processing unit(CPU): This piece of hardware is used to direct and perform most of the computation in the computer. Depending on the varaety of motherboard, the variety of CPUs compatible with the motherboard will vary.

RAM(temporary memory): The RAM chips, or local memory chips, record basic memory, so that you can quickly upload something you recently used, or visited. RAM is what allows you to quickly load the last tab you closed on a chromebook. The Ram chips are installed on the motherboard, with up to four chips.

Hard Drive: The Hard drive of a pc stores all the things that you download, such as games, pictures, even your computer settings, such as wallpaper.

Power Supply: This piece of hardware takes ower from places such as outlets and brings it to the computer.

Component communication: The components in a pc communicate through the circuts in the motherboard and other components.

Keyboard, Mouse, and Monitor: These components allow for user imput, as well as visual representation of said user input. The keyboard allows for users to create text, and perform other actions with the keys. The mouse allows the user to select things, and to move objects on the screen. The monitor is used to show a visual representation of the user input, and the objects on the screen.
